# 概述：
>1.基于op库(https://github.com/WallBreaker2/op) 运用于阴阳师桌面端的自动化脚本，致力于为各位痒痒鼠大人清寮突破！
>
>2.该脚本可自动调整桌面端阴阳师窗口大小，有基于tkinter写的简易ui
>
>3.该脚本因后台图色不稳定容易报错，所以仅鼠标设置为后台模式
>
>4.该脚本仅供学习交流
